title: 'CraftEngineFurniture: place and manage furniture'
description: 'API reference for CraftEngineFurniture — methods to place, remove, query, and ray-trace furniture entities using the CraftEngine Java API.'
---

`CraftEngineFurniture` is a final utility class in `net.momirealms.craftengine.bukkit.api` that lets you spawn, remove, look up, and ray-trace furniture entities. Every method is `static`. All world-mutating calls must be made from the thread that owns the relevant region (Folia compatibility).

```java
import net.momirealms.craftengine.bukkit.api.CraftEngineFurniture;
import net.momirealms.craftengine.bukkit.entity.furniture.BukkitFurniture;
import net.momirealms.craftengine.core.entity.furniture.CustomFurniture;
import net.momirealms.craftengine.core.entity.furniture.Furniture;
import net.momirealms.craftengine.core.util.Key;
```

## loadedFurniture()

```java
@NotNull
public static Map<Key, CustomFurniture> loadedFurniture()
```

Returns an unmodifiable map of all currently loaded custom furniture definitions. The map keys are unique `Key` identifiers and the values are the corresponding `CustomFurniture` instances.

<Warning>
  Do not call this method during `onEnable()`. The map will be empty until CraftEngine finishes loading its content. Listen for `CraftEngineReloadEvent` and access this method from within that handler.
</Warning>

**Returns** a non-null, unmodifiable map of all loaded custom furniture.

```java
@EventHandler
public void onReload(CraftEngineReloadEvent event) {
    Map<Key, CustomFurniture> furniture = CraftEngineFurniture.loadedFurniture();
    getLogger().info("Loaded " + furniture.size() + " furniture type(s).");
}
```


Gets a custom furniture definition by its unique identifier.

**Parameters**

| Name | Type | Description |
|---|---|---|
| `id` | `Key` | The unique namespaced key of the custom furniture |

**Returns** the `CustomFurniture` for the given ID, or `null` if no furniture with that ID is loaded.


## rayTrace(Player, double)

```java
@Nullable
public static BukkitFurniture rayTrace(Player player, double maxDistance)
```

Performs a ray trace from the player's eye position to find the furniture entity they are looking at, up to the given maximum distance.

**Parameters**

| Name | Type | Description |
|---|---|---|
| `player` | `Player` | The player performing the ray trace |
| `maxDistance` | `double` | Maximum distance in blocks to trace |

**Returns** the `BukkitFurniture` the player is targeting, or `null` if none is found within range.

```java
BukkitFurniture targeted = CraftEngineFurniture.rayTrace(player, 8.0);
if (targeted != null) {
    player.sendMessage("Looking at furniture: " + targeted.id());
}
```

## place(Location, Key)

```java
@Nullable
public static BukkitFurniture place(Location location, Key furnitureId)
```

Places furniture at a location using the furniture definition's first available variant. No sound is played.

**Parameters**

| Name | Type | Description |
|---|---|---|
| `location` | `Location` | The location to spawn the furniture |
| `furnitureId` | `Key` | The unique ID of the furniture definition |

**Returns** the spawned `BukkitFurniture`, or `null` if no furniture with that ID exists.

```java
BukkitFurniture chair = CraftEngineFurniture.place(
    location,
    Key.of("mypack", "oak_chair")
);
```

## place(Location, CustomFurniture, String, boolean)

```java
@NotNull
public static BukkitFurniture place(
    Location location,
    CustomFurniture furniture,
    String variant,
    boolean playSound
)
```

Places furniture using a `CustomFurniture` definition object directly, with variant and sound control. Use this overload when you already hold a reference to the definition.

**Parameters**

| Name | Type | Description |
|---|---|---|
| `location` | `Location` | The location to spawn the furniture |
| `furniture` | `CustomFurniture` | The furniture definition |
| `variant` | `String` | The variant name to use |
| `playSound` | `boolean` | Whether to play the furniture's place sound |

**Returns** a non-null `BukkitFurniture` representing the spawned entity.

```java
CustomFurniture def = CraftEngineFurniture.byId(Key.of("mypack", "bookshelf"));
if (def != null) {
    BukkitFurniture spawned = CraftEngineFurniture.place(
        location, def, def.anyVariantName(), true
    );
}
```

## place(Location, CustomFurniture, CompoundTag, boolean)

```java
@NotNull
public static BukkitFurniture place(
    Location location,
    CustomFurniture furniture,
    CompoundTag data,
    boolean playSound
)
```

Places furniture using a full NBT `CompoundTag` as the data accessor, giving you low-level control over every piece of furniture data including variant, rotation, and custom properties.

**Parameters**

| Name | Type | Description |
|---|---|---|
| `location` | `Location` | The location to spawn the furniture |
| `furniture` | `CustomFurniture` | The furniture definition |
| `data` | `CompoundTag` | An NBT compound tag containing the furniture data |
| `playSound` | `boolean` | Whether to play the furniture's place sound |

**Returns** a non-null `BukkitFurniture` representing the spawned entity.

```java
CompoundTag data = new CompoundTag();
data.putString("variant", "north");
data.putFloat("yaw", 90f);

CustomFurniture def = CraftEngineFurniture.byId(Key.of("mypack", "sign_post"));
if (def != null) {
    BukkitFurniture spawned = CraftEngineFurniture.place(location, def, data, true);
}
```

## Deprecated place overloads

<Warning>
  The following `place` overloads that accept `AnchorType` are **deprecated since 0.0.66 and will be removed** in a future version. Replace them with the equivalent `String variant` overloads.

  - `place(Location, Key, AnchorType)` → use `place(Location, Key, String)`
  - `place(Location, Key, AnchorType, boolean)` → use `place(Location, Key, String, boolean)`
  - `place(Location, CustomFurniture, AnchorType)` → use `place(Location, CustomFurniture, String, boolean)`
  - `place(Location, CustomFurniture, AnchorType, boolean)` → use `place(Location, CustomFurniture, String, boolean)`
</Warning>

## remove(Entity)
