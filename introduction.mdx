CraftEngine is a next-generation Paper and Folia plugin that enables server administrators and developers to dynamically create custom content — blocks, items, furniture, recipes, and more — entirely through configuration files. No client-side mods are required; everything is delivered via an automatically generated and hosted resource pack.

## What CraftEngine can do

CraftEngine repurposes vanilla Minecraft block states and packet interception to deliver fully custom content without requiring players to install any mods. The plugin generates, validates, optimizes, and delivers a resource pack automatically, so players receive textures and models the moment they join.

## Key features

- **No client mods required** — custom content is delivered via resource pack
- **Automatic resource pack generation** — textures, models, atlases, fonts, and sounds are assembled and served automatically
- **Folia support** — fully compatible with multi-threaded Folia servers
- **Template system** — reuse block states, models, loot tables, and event definitions across multiple content files
- **Plugin interoperability** — a Java API with Bukkit events lets other plugins interact with custom blocks, items, and furniture
- **Premium features** — resource pack obfuscation, client-side model binding, entity culling, and more in the Premium Edition
